# `netlify/functions/flashagent-download.ts`

> Serves CSV exports generated by Flashi during chat flows.

## Exports

| Name | Type | Description |
|------|------|-------------|
| `handler` | `(request: Request, _context: Context) => Promise<Response>` | Default Netlify function handler |

## Dependencies (imports from project)

| Import | From | Used for |
|--------|------|----------|
| `requireAuth` | `_lib/auth` | Authentication |
| `requireEnvironmentAccessScopeForResourcePermission` | `_lib/rbac` | `flashagent:read` environment permission check |
| `queryOne` | `_lib/db` | Resolve environment workspace |
| `getBlob` | `_lib/blobs` | Read generated CSV content |
| `getSearchParams`, `errorResponse` | `_lib/helpers` | Query parsing and error responses |

## Key Logic

GET-only endpoint with required query params:

- `id` (CSV export UUID)
- `environment_id` (UUID)
- optional `filename`

Flow:

1. Authenticate request.
2. Validate `id` and `environment_id` as UUIDs.
3. Resolve environment workspace and enforce `flashagent:read` access for that environment.
4. Load blob key `flashagent/{workspace_id}/{environment_id}/{id}.csv` from `exports` store.
5. Return CSV as attachment (`Content-Disposition`) with sanitized filename.

## API Surface

| Method | Route | Auth | Description |
|--------|-------|------|-------------|
| `GET` | `/api/flashagent/download` | Session or API key (`flashagent:read`) | Download a CSV generated by Flashi |
