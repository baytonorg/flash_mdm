# Tech stack

This page is a quick reference for reviewers.

## Frontend
- React 19 (SPA, lazy-loaded routes)
- Vite
- TypeScript
- Tailwind CSS v4
- TanStack Query (data fetching/caching)
- TanStack Table (data grids)
- Zustand (global state)
- Monaco Editor (policy JSON editing)
- React Hook Form + Zod (forms and validation)
- React Router v7
- Lucide React (icons)
- Chart.js / react-chartjs-2 (analytics)

## Backend
- Netlify Functions (Node.js, ESM)
- TypeScript

## Data
- Postgres (via `pg`)
- Netlify Blobs (object storage)

## External services
- Google Android Management API (AMAPI)
- Google Pub/Sub (AMAPI push notifications)
- Google Maps JavaScript API (geofencing UI)
- Stripe (billing/licensing)
- Resend (transactional email)

## Testing
- Vitest
- Testing Library (React)
- jsdom
